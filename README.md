# Home Assistant Add-ons

This repository contains custom Home Assistant add-ons developed by gabrielgbim.

## Add-ons

### Nest Video Downloader

Download videos from your Nest connected cameras directly to your Home Assistant instance.

[View add-on documentation](nest_downloader/README.md)

## Installation

To use these add-ons:

1. Add this repository to your Home Assistant instance:
   - Go to **Settings** → **Add-ons** → **Add-on Store**
   - Click the three dots menu (⋮) in the top right
   - Select **Repositories**
   - Add: `https://github.com/gabrielgbim/ha-addons`
   
2. Install the desired add-on from the add-on store
3. Configure the add-on options
4. Start the add-on
